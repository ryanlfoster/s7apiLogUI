CUI.Modal

A dialog that prevents interaction with page elements while displayed. Modal will use existing markup if it is present, or create markup if options.element has no children.

Example

Show Modal

A Sample Modal

Some sample content.

Code

Instantiate with Class

var modal = new CUI.Modal({
  element: '#myModal',
  heading: 'My Modal',
  content: '<p>Content here.</p>',
  buttons: [
    {
      label: 'Save',
      className: 'primary',
      click: function(evt) {
        console.log('Modal: This would usually trigger a save...');
        this.hide(); // could also use evt.dialog.hide();
      }
    }
  ]
});

// Hide the modal, change the heading, then show it again
modal.hide().set({ heading: 'My Modal Again'}).show();

// jQuery style works as well
$('#myModal').modal('hide');

Instantiate with jQuery

$('#myModal').modal({
  heading: 'My Modal',
  content: '<p>Content here.</p>',
  buttons: [
    {
      label: 'Close',
      click: 'hide', // Specifying 'hide' causes the dialog to close when clicked
    }
  ]
});

// Hide the modal, change the heading, then show it again
$('#myModal').modal('hide').modal({ heading: 'My Modal Again'}).modal('show');

// A reference to the element's modal instance is stored as data-modal
var modal = $('#myModal').data('modal');
modal.hide();

Data API: Instantiate and show modal

When using a <button>, specify the jQuery selector for the element using data-target. Markup should exist already if no options are specified.

<button data-target="#myModal" data-toggle="modal">Show Modal</button>

Data API: Instantiate, set options, and show

When using an <a>, specify the jQuery selector for the element using href. Markup is optional since options are specified as data attributes.

<a 
  href="#modal"
  data-toggle="modal"
  data-heading="Test Modal"
  data-content="&lt;p&gt;Test content&lt;/p&gt;"
  data-buttons='[{ "label": "Close", "click": "close" }]'
>Show Modal</a>

Data API: Instantiate, load content asynchronously, and show

When loading content asynchronously, regardless of what tag is used, specify the jQuery selector for the element using data-target and the URL of the content to load with href.

<button
  data-target="#myModal"
  data-toggle="modal"
  href="content.html"
>Show Modal</button>

Markup

<div id="myModal" class="modal">
  <div class="modal-header">
    <h2>Heading</h2>
    <button type="button" class="close" data-dismiss="modal">&times;</button>
  </div>
  <div class="modal-body">
    Content
  </div>
  <div class="modal-footer">
    <button data-dismiss="modal">Close</button>
  </div>
</div>

Markup with <form> tag

Modals can be created from the <form> tag as well. Make sure to set type="button" on buttons that should not perform a submit.

<form id="myModal" class="modal" action="/users" method="post">
  <div class="modal-header">
    <h2>Create User</h2>
    <button type="button" class="close" data-dismiss="modal">&times;</button>
  </div>
  <div class="modal-body">
    <label for="name">Name</label><input id="name" name="name" type="text">
  </div>
  <div class="modal-footer">
    <button type="button" data-dismiss="modal">Cancel</button>
    <button type="submit">Submit</button>
  </div>
</form>

CUI.Modal

A dialog that prevents interaction with page elements while displayed. Modal will use existing markup if it is present, or create markup if options.element has no children.

Example

Show Modal

A Sample Modal

Some sample content.

Code

Instantiate with Class

var modal = new CUI.Modal({
  element: '#myModal',
  heading: 'My Modal',
  content: '<p>Content here.</p>',
  buttons: [
    {
      label: 'Save',
      className: 'primary',
      click: function(evt) {
        console.log('Modal: This would usually trigger a save...');
        this.hide(); // could also use evt.dialog.hide();
      }
    }
  ]
});

// Hide the modal, change the heading, then show it again
modal.hide().set({ heading: 'My Modal Again'}).show();

// jQuery style works as well

Instantiate with jQuery

$('#myModal').modal({
  heading: 'My Modal',
  content: '<p>Content here.</p>',
  buttons: [
    {
      label: 'Close',
      click: 'hide', // Specifying 'hide' causes the dialog to close when clicked
    }
  ]
});

// Hide the modal, change the heading, then show it again

// A reference to the element's modal instance is stored as data-modal

modal.hide();
