Code

Instantiate with Class

var tabs = new CUI.Tabs({
  element: '#myTabs',
  tabs: [
    {
      id: 'first-tab'
      label: 'First Tab',
      content: 'First tab content here!',
      active: true
    },
    {
      id: 'second-tab',
      remote: 'foobar.html',
      label: 'Remote tab'
    }
  ]
});

// Hide the tabs, set the active tab, and show it again
tabs.hide().set({active: 'second-tab'}).show();

// jQuery style works as well
$('#myTabs').tabs('show');

Instantiate with jQuery

$('#myTabs').tabs({
  type: 'stacked'
  tabs: [
    {
      id: 'first-tab'
      label: 'First Tab',
      content: 'First tab content here!',
      disabled: true
    },
    {
      label: 'Second tab'
      content: 'Without an ID, this tab can be addressed by its index (1).',
      active: true
    }
  ]
});

// A reference to the element's tabs instance is stored as data-tabs
var tabs = $('#myTabs').data('tabs');
tabs.hide();

Data API: Instantiate, set options, and show

ou do not need to explicitly instantiate a tabs instance to use the tabs functionality. The data API will handle switching between tabs as long as you have created a <div> with the tabs class. When using markup to instantiate tabs, the overall container is div class="tabs". The tabs themselves are specified within the nav block as simple a elements. The data-toggle="tab" attribute on a nav links is essential for the data API; do not omit. The href can either be an id of a following section, a simple anchor: #, or a remote link (see next example).

<div class="tabs">
  <nav>
    <a href="#" data-toggle="tab" class="active">Tab 1</a>
    <a href="#second_tab" data-toggle="tab">Tab 2</a>
  </nav>
  <section class="active">Lorizzle ipsizzle fo shizzle mah nizzle fo rizzle.</section>
  <section id="second_tab">Nulla gangsta. Brizzle shizzlin dizzle pharetra neque. </section>
</div>

Data API: Instantiate, load content asynchronously, and show

When loading content asynchronously, specify the jQuery selector for the element using data-target and the URL of the content to load with href.

<div class="tabs">
  <nav>
    <a href="remote.html" data-target="#tab1" data-toggle="tab" class="active">Remote Tab 1</a>
    <a href="remote_dos.html" data-target="#tab2" data-toggle="tab">Remote Tab 2</a>
  </nav>
  <section id="tab1" class="active"></section>
  <section id="tab2"></section>
</div>

Variants

The possible variants, white, stacked, and nav, are specified either via the type argument to the constructor, or via manually specifying the class alongside tabs.

<div class="tabs nav">
  <nav>
    <a href="#" data-toggle="tab" class="active">Menu Item One</a>
    <a href="#" data-toggle="tab">Menu Item Two</a>
    <a href="#" data-toggle="tab" class="disabled">Disabled Menu Item</a>
  </nav>
  <section class="active">Lorizzle ipsizzle fo shizzle mah nizzle fo rizzle.</section>
  <section>Nulla gangsta. Brizzle shizzlin dizzle pharetra neque. </section>
  <section>This section will never be shown :(</section>
</div>
