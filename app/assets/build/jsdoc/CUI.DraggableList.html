CUI.DraggableList

Makes a list draggable. This also implies re-ordering of the list items on behalf of the user. This widget also defines some high-level events for inserting, reordering and removing items. The underlying drag/drop events should not be used outside this widget, as they are subject to change in the future.

Examples

Code

A fully configured list

      <div class="dropzone">
        <ul class="draggable" data-init="draggable-list" data-allow="drag drop reorder" data-closeable="true">
        <li>Item 1</li>
        <li>Item 2</li>
        </ul>
        <p>Drop here!</p>
      </div>
