Using CoralUI in Your Project

Getting CoralUI in your project is simple.

Any Project

Do a full build with grunt full and add the contents of the build/ directory to your project. See the Build Process & Technology wiki page for more information.

Granite Projects

Use grunt mvn to install to your local Granite instance. See the CoralUI in Granite wiki page for more information.
