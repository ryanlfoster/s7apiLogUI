Base

The abstract class which contains methods that all classes will inherit. Base cannot be extended or instantiated and does not exist in the global namespace. If you create a class using new Class() or MyClass.extend(), it will come with Base' methods.

Constructor

Base is an abstract class and cannot be instantiated directly. Constructors are chained automatically, so you never need to call the constructor of an inherited class directly
